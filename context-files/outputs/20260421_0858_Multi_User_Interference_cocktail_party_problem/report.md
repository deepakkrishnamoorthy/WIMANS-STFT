# Experiment Report: 20260421_0858_Multi_User_Interference_cocktail_party_problem

## Summary
Visualized the 'Cocktail Party Problem' by plotting STFT spectrograms for 1, 3, and 5 users. As the number of users increases, the micro-Doppler signatures become highly entangled and the noise floor rises significantly. This proves that simple models will fail in crowded environments; we must utilize advanced source-separation architectures (like deep Transformers or Attention mechanisms) for multi-user subsets.
